Actualización del repositorio (main)

He subido cambios nuevos a la rama main. Por favor actualicen sus repositorios locales:

  1. git checkout main
  2. git pull origin main

Si están trabajando en su rama, actualícenla con:

  1. git checkout "Leandro Maquetado"

Pasos para crear y mezclar la rama leandro

1️⃣ Asegúrate de estar en main y actualizado:

git checkout main
git pull origin main

2️⃣ Crear la nueva rama y entrar en ella:

git checkout -b leandro

3️⃣ Guardar los cambios:

git add .
git commit -m "Cambios realizados por Leandro"

4️⃣ Subir la rama al repositorio remoto:

git push origin leandro

5️⃣ Mezclar con main:

git checkout main
git pull origin main
git merge leandro

6️⃣ Subir merge al repositorio:

git push origin main
